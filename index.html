javascript的实现

1.javascripi实现

javascript的实现分为三个部分 一个是ecmascript,一个是dom（document object model),和BOM(browser object model)。其中ecmascript为其语言核心，拥有着基本的数据类型，语法属性方法等。DOM是javascript用于文档结构的操作，操纵网页内容。Bom是javascript对浏览器窗口的操作，与浏览器进行交互。

排序算法总结(三)归并排序

1归并排序

给定一数组nums，随机打算元素顺序，循环从中间将数组一份为二，直至最小单位为止。将每两个小集合按照有序的方式插入大集合中，最终递归至最大集合，至此排序完成。

2快速排序

给定一数组nums，随机打乱元素顺序，选定元素nums[0]，声明头尾两指针，头指针递增直至遇到某元素大于选定元素，尾指针递减直至遇到某元素小于选定元素，交换头尾指针指定元素的位置。当头尾指针相等时，将头指针与nums[0]交换。循环直至排序完成。

排序算法总结(二)插入排序

插入排序

给定一无序数组nums，指针从第0位开始遍历数组，比较nums[i]和nums[i+1]的大小，如果nums[i]>nums[i+1]，交换两元素顺序。每循环一次后，i递增1。循环次数最多为数组nums的长度次。时间复杂度On=n^2,空间复杂度On=1。

选择排序

概述
